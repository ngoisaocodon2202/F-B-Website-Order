## 🎯 MỤC TIÊU DỰ ÁN
Xây dựng **website đặt món ăn online** giúp người dùng:
- Xem danh sách món ăn theo danh mục  
- Thêm món vào giỏ hàng và đặt món  
- Đăng nhập, thanh toán đơn hàng  
- Giúp quán ăn dễ dàng quản lý menu và đơn hàng
